Haley Sulcer is a web developer
based in the New York area.
PROJECTS:
PC Wizard - I built a resource to help people build
computers. Users can view information about different
types of computer components, as well as information
regarding specific computer parts (series, price,
form factor, etc). Built with: Rails, Devise, and Bulma.
Spark Lines - I created a single-page news aggregator
in which new posts appear on page refresh and people can
submit their own articles.
Built with: Express, Node, PostgreSQL, and Newsapi.
Terminal Hacker - I recreated the terminal hacking game
from the Fallout series. Randomly generated words
are spawned on the page and the user has to
figure out which word the computer selected.
Built with: HTML, CSS, and JS.
Marvel Character Finder - My group and I used
Marvel's API on this project. Users can create accounts,
view information on over a thousand characters,
add characters to their favorites list, and view their
favorites on a separate page.
Built with: Marvel API, React, Express, Node,
PostgreSQL, and Bootstrap.
